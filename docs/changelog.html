v1.0

Initial Release
Added
  • Folder-based link organization
  • Built-in yt-dlp powered downloader with background queue system
  • YouTube metadata scraping — thumbnail embedded as album art, artist and title embedded as ID3 tags in downloaded MP3
  • Auto-naming on share sheet import — app scrapes title, artist, and song name when a link is shared into it
  • Manual rename support for any link
  • ZIP import and export system for sharing complete folder collections
  • Cross-folder search by artist name or song title from the main screen
  • Available in four APK builds: arm64-v8a, armeabi-v7a, x86_64, x86
